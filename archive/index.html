Archive backup guide
  1. Copy the original chat log range from the spreadsheet.
  2. Open Chat Log Converter with the + button.
  3. Paste the rows, check title, category, date, and colors, then convert.
  4. Paste the full output into _chats/your-file.html.
  5. Put images in assets/archive/log-folder/ and use matching image paths.
  6. Commit and push with GitHub Desktop.
