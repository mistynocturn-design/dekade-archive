Twitter backup guide
  1. Copy spreadsheet columns A:F: date, author, profile image, tags, content, thread ID.
  2. Open Twitter Archive Converter with the + button.
  3. Check Month title, Archive sort date, Note font size, and Author styles.
  4. Paste the rows and convert.
  5. Paste the full output into _twitter/YYYY-MM.md.
  6. Put profile images in assets/twitter/YYYY-MM/ and match the spreadsheet paths.
  7. Commit and push with GitHub Desktop.
