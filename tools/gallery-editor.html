Gallery Editor

사용방법
  1. 이미지를 repo의 assets/gallery/작품폴더/에 먼저 넣습니다.
  2. 작품 정보와 이미지 경로를 입력하고 Add image로 묶음에 추가합니다.
  3. Current images에서 순서를 바꾸거나 삭제하고 Preview를 확인합니다.
  4. Save draft는 이 브라우저에만 임시 저장합니다.
  5. 완성된 Markdown을 Download한 뒤 repo의 _gallery 폴더에 넣고 Commit/Push합니다.
