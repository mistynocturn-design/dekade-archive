Checklist Editor

PDF 체크리스트 한 항목의 메타데이터와 저장 파일을 만듭니다.

사용 방법 · 저장 경로
  1. PDF 저장 폴더 연결을 누르고 dekade-archive 저장소와 assets/checklists/체크리스트-폴더를 차례로 선택합니다.
  2. 연결 완료 메시지를 확인한 뒤 PDF 선택을 눌러 PDF를 고릅니다.
  3. 선택한 파일은 해당 폴더에 sheet1.pdf, sheet2.pdf처럼 다음 번호로 저장되고 PDF path가 자동 입력됩니다.
  4. 완성된 Markdown은 _checklists/영문-파일명.md에 넣습니다.
