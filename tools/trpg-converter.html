TRPG Log Converter

이용방법
  1. 코코포리아에서 내보낸 HTML 로그를 불러옵니다.
  2. 세션 정보와 캐릭터별 표시 이름·닉네임색·프로필 이미지를 설정합니다.
  3. Current items에서 메시지를 선택해 오타를 고치거나 이동·삭제하고, 대사 또는 이미지를 원하는 위치에 추가합니다.
  4. Preview로 확인한 뒤 Markdown을 내려받아 repo의 _trpg 폴더에 넣습니다.
  5. 표지·프로필·삽입 이미지는 먼저 assets/trpg/세션폴더/에 넣고 해당 경로를 입력합니다.
