Writing Editor

이용방법
  1. 원문이 있다면 Original URL을 먼저 입력합니다.
  2. 글 정보와 Markdown 본문을 작성합니다. 입력 내용은 이 브라우저에 자동 저장됩니다.
  3. 이름 있는 초안은 Save draft로 별도 보관할 수 있습니다.
  4. 완성된 파일을 Download한 뒤 repo의 _writings 폴더에 넣고 Commit/Push합니다.
  5. 표지를 사용한다면 파일을 assets/writings/에 먼저 넣습니다.
