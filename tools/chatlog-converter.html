Chat Log Converter

사용방법
  1. 최초 1회: 스프레드시트의 확장 프로그램 → Apps Script에서 repo의 appscript/archive_twitter_Code_gs.txt 내용을 붙여넣습니다.
  2. Apps Script의 스크립트 속성에 BACKUP_READ_KEY를 만들고 원하는 비밀 문자를 저장한 뒤, 웹 앱을 '나로 실행 / 모든 사용자'로 배포합니다.
  3. 배포된 /exec URL과 방금 정한 Read key를 아래에 입력합니다. 같은 값은 Twitter 컨버터에서도 사용하며 이 브라우저가 기억합니다.
  4. 스프레드시트의 Archive 탭에 대화를 정리합니다.
  5. 시작 행과 끝 행을 입력하고 Load rows를 누릅니다. 같은 제목 전체 확장을 켜면 범위 양끝의 이어진 대화까지 포함합니다.
  6. 색상과 제목을 확인한 뒤 Convert를 누르고, 결과물을 _archive/파일명.html에 저장합니다.
  7. 이미지는 미리 repo의 assets/archive/...에 넣고 시트의 이미지 열에 같은 경로를 적습니다.

Load from spreadsheet

Expected columns: date, title, text, image, side. This converter outputs finished bubble HTML, so multiline text and spreadsheet quotes do not need to be parsed by the site.
