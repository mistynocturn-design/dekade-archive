Backup workflow
  1. LIVE 게시판에서 대화 타래를 완료 상태로 바꿉니다.
  2. Reload 후 해당 타래를 선택하고 Load thread를 누릅니다.
  3. 제목, 카테고리, 날짜와 파일명을 확인한 뒤 Download package를 누릅니다.
  4. ZIP의 _roleplays 안 새 HTML만 repo의 같은 폴더에 추가합니다.
  5. ZIP의 이미지 하위 폴더를 repo의 assets/roleplay에 추가합니다.
  6. Commit/Push 후 Basic, AU 또는 ETC에서 정상 표시되는지 확인합니다.
  7. 확인이 끝난 뒤에만 Mark archived를 누릅니다.

.gitkeep과 기존 대화 파일은 지우거나 덮지 않아도 됩니다. 새 파일만 추가하면 됩니다.
