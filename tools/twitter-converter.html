Twitter Archive Converter

사용방법
  1. 최초 1회: 스프레드시트의 확장 프로그램 → Apps Script에서 repo의 appscript/archive_twitter_Code_gs.txt 내용을 붙여넣습니다.
  2. Apps Script의 스크립트 속성에 BACKUP_READ_KEY를 만들고 원하는 비밀 문자를 저장한 뒤, 웹 앱을 '나로 실행 / 모든 사용자'로 배포합니다.
  3. 배포된 /exec URL과 방금 정한 Read key를 아래에 입력합니다. 같은 값은 Archive 컨버터에서도 사용하며 이 브라우저가 기억합니다.
  4. 스프레드시트의 Twitter 탭에 트윗을 정리합니다.
  5. 원트윗 기준 월을 선택하고 Load month를 누릅니다. 다음 달까지 이어진 타래도 시작월 파일에 함께 포함됩니다.
  6. 타래ID는 월마다 1부터 다시 써도 됩니다. 같은 타래의 행은 스프레드시트에서 서로 붙여 둡니다. 여러 달에 걸친 타래는 선택 사항인 타래시작월 열을 추가하면 가장 정확합니다.
  7. 작성자 색상과 미리 생성된 결과를 확인한 뒤 _twitter/YYYY-MM.md로 저장합니다.
  8. 프로필·첨부 이미지는 repo에 올린 뒤 시트에 해당 경로를 기록합니다.

Load from spreadsheet

One author per line: author|box background|text color. Names must match the spreadsheet author column.

Expected columns: date, author, profile image, tags, content, image URLs, thread ID. Thread start month is optional. Put 1 to 4 image URLs in the image cell, one URL per line.
