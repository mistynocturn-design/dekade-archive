World Archive Editor

World 포털, 설정 문서, AU 레지스트리를 한 프로젝트에서 관리합니다.

사용 방법 · 파일별 저장 경로
  1. 이미지부터 연결: 이미지 저장 폴더 연결을 누르고 먼저 dekade-archive 저장소 폴더, 다음으로 그 안의 assets/world/원하는-폴더를 선택합니다. 새 이미지는 선택한 폴더에 01.png, 02.jpg 순으로 바로 저장되고 입력칸과 미리보기에 자동 반영됩니다.
  2. Portal: 편집 후 world.yml을 내려받아 저장소의 _data/world.yml과 교체합니다.
  3. Documents: Foundation은 world/foundation.md, Chronicle은 world/chronicle.md, Archive는 world/archive.md로 저장합니다. Custom 문서는 world/파일이름.md에 넣고 연결 URL도 직접 추가해야 합니다.
  4. AU Registry: 내려받은 AU Markdown은 _world_aus/파일이름.md에 넣습니다. 파일명은 영문 소문자와 하이픈 사용을 권장합니다.
  5. Project JSON: 에디터 작업 백업 전용입니다. 사이트 폴더에 넣지 말고 개인 백업 위치에 보관한 뒤 Load JSON으로 이어서 작업합니다.
  6. 기존 이미지 버튼을 누르면 연결한 폴더 안의 이미지를 골라 경로만 다시 사용할 수 있습니다. 폴더 밖 이미지를 고르면 새 번호로 복사됩니다.
